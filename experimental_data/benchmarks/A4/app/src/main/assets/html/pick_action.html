The interaction with Android's telephony service has been severly restricted with the latest Android major releases. This has been motivited mostly by security reasons. In order for the app to run on different versions and devices we offer different actions to choose when an incoming event occurs. Some are guaranteed to work on all devices, like logging or sending a warning. Other are device/version dependent.

When choosing the action to perform one has to do some experimentation by calling the device from known sources and finding the action that works. If the device is rooted, there is some confidence that DropCallByRoot should work. The first time it is used it will ask for a root privilege to be granted.

Our suggestion is to try first with DropCallByEndCall. If it does not work try DropCallByDownButton followed by DropCallByUpButton.

In future versions of this app, more actions will be added to try to keep the app functioning properly for a high percentage of devices and versions.
