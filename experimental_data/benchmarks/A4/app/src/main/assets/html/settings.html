The setting activity helps customizing the usage of this app. Here is a list of the setting that can be changes:

  • Max size of DB query determines the number of most recent rows to be returned from a DB query. This can be limited to a small number to view the most recent logs, or can be unlimited to access the full log. It affects the logs of service runs, logged calls and triggered events
  • Keep call logs for gives the longevity of the logs. After the selected time the logs are purged
  • Show toast messages determines the verbosity of toast messages
  • Export rules allows to export all the rules in a binary format to a fixed location, shown below, and available on all devices under the Download directory. This can be used as an extra store of rules when upgrading your device
  • Import rules allows to re-import all the rules in binary format from a fixed location, shown below, and available on all devices under the Download directory. This can be used as an extra store of rules when upgrading your device
  • Show protected app dialog: certain devices, e.g. the Huawei recent ones, keep stopping the app when the screen goes off. It is possible to add this app to a protected list. A dialog should appear on the first usage of the app to ask for this permission, and then this dialog can be disabled. This setting allows to remind the app to re-enable this dialog again.
