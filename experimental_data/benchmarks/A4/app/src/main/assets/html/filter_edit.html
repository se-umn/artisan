Filters are used to customize the actions to be performed on incoming calls. Based on the time and date of the call and the incoming number certain actions can be triggered, like dropping a call, logging it or else. Hence a filter is determined by the calendar rule it uses, the filter rule that matches the incoming number and the action to be performed.

Filters can be combined and they are acted sequentially. Typically one or two filters are necessary and all the filters currently defined by the user will be used. So make sure to remove undesired filters, if they have been created only for testing purposes and are not needed.

Filters can be created, updated, edited and deleted and these are the icons for the editing actions and their meanings:

  • for saving or updating a filter
  • for changing (editing) an existing filter
  • for undoing changes and returning to the original filter
  • for deleting an existing filter
  • for adding a new filter

Once in edit or create mode, one can use the menu items in the action bar to perform the following:

  • Pick a calendar rule from the list of existing or create a new one
  • Pick a filter rule from the list of existing or create a new one
  • Pick an available action to be performed
  • Edit the currently selected calendar rule
  • Edit the currently selected filter rule.
